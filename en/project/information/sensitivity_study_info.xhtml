Sensitivity Study Information

A sensitivity study is done by the simulation tool. The sensitivity study compares the current building design with the optimal building design in the given climate. Important design parameters will be compared separately so that the user can see the influence of each design parameter. The bar diagrams are illustrating the difference between the optimal building and the current building. Negative values are illustrating that the current parameter setting is better then the optimal reference case of the simulation tool. These parameters should not be changed. The software provides a recommendation which design parameter should be changed. Recommended parameters are marked with a green sign

The calculation of the sensitivity study can take several minutes because it involves a huge number of simulations. It involves also the calculation of an optimization algorithm
